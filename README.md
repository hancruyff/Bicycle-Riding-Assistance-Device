## 1. 개요

자전거 주행 보조 장치 개발

## 2. 주요 기능

- 기울어지는 방향으로 핸들 조정 기능
- 어두운 곳에서 자동 조명 기능
- 사고 발생 시 미리 입력된 연락처로 연락 기능

## 3. 구현

기울기 센서를 통해 기울어 지는 방향과 기울기 감지
기울어지는 방향으로 모터를 이용하여 핸들을 조정


조도 센서를 이용하여 어두워질 경우 led조명 작동



충격 센서를 이용하여 일정 수준 이상의 충격이 가해질 경우 
사고가 발생한 것으로 감지 
미리 등록된 연락처로 메시지 전송
